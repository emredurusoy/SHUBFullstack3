Css Eğitimi

Css Nedir?

Css : casding style sheet in kısaltmasıdır Ekrandaki html ögelerinin tasarımlarını değiştirmemizi sağlar

Css Yazma Seçenekleri

  1. Inline : Html elementlerinin içerisne style attrivute ü açılarak yazılır
  2. Sayfa head alanı içerisnde style etiketleri arasında yazılarak
  3. Harici bir css oluşturup sayfa head alanı içinde çağırarak.

Inline Css Yazılımı

elementlerini açılış taglarına bırakıp stlye özleiği içine css yazarak yapırız
eğer birden fazla css özlliği uygulayacaksak özellikler arasına ; koyarız
En sondakine ; koymaya gerek yoktur

Head Tagları Arasına Css Yazımı

Sayfa head alanı içerisnde style etikeltri arasına yazılarak sayfadaki elementlere stil uygulanabilr.

Harici Css Dosyası Kullanarak Tasarım Uygulama

Harici bir css dosayası oluşturup (stlye.css gibi) sayfa head alanı içinde link özelliği ile çağırarak sayfa tasarımını şekillendirebilriz

Css Seçiciler

1 - Element Seçiciler

Sayfadaki elementleri html taglarına göre (p, h1, div, vb) seçerek css uygulayabiliriz
Bu durumda sayfalardaki tüm elementler seçilecekletir

2 - Id seçici

Sayfalardaki elemnetleri id özelliğine göre seçip stil uygulammazı sağlar
Genelede sayfalaradaki tek bir elementi seçmek için kullanırız
Css dosyası içerisdnen # karakteri yazılıp elementin id değri eklenerek element seçilir
Örnek: #idSecici{}

3-Class-Sınıf Seçici

Web sayfasında veya uyhulama genelinde birden fazla elemente ayı stil ayarlarını uygulamak için class seçicileri kullanırız

Element tagi içiersinde class yazıp css dosyasında yazılan class adını vererek kullanırız

Css dosyası içerinde . işaretiyle class oluştururuz
örneğin: .sinifSecici{}

Bir elemente class özelliği içerisine aralarına boşluk bırakarak birden fazla class uygulayabiliriz.

Ana div içindeki ilk p

Bir html elementinin içindeki alt elementleri hedefleyerek de stil uygulayabiliriz.

Bunun için ana elementi seçip boşluk veya > işaretiyle alt elmentleri seçebiliyoruz. Örneğin; #anaDiv p{}

Ana div içindeki son p
